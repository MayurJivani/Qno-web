

# QNO – Quantum-Themed Multiplayer Card Game

QNO is a web-based real-time multiplayer card game designed to teach fundamental quantum computing concepts through interactive gameplay.

The game models principles such as quantum gates, superposition, and entanglement using an intuitive card-based system.

🌐 Live Demo: [https://qno-web.web.app](https://qno-web.web.app)
